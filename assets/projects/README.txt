HOW TO ADD YOUR PROJECT SCREENSHOTS
===================================

1. Open this folder in File Explorer:
   C:\Users\ranji\.cursor\projects\empty-window\assets\projects

2. Copy your screenshot files here.

3. Rename each file EXACTLY like this (use .png or .jpg):

   dhwani-ai.png
   grievance.png
   thailavardhana.png
   heart-disease.png
   tele-seva.png
   crime-investigation.png
   cricket-pro.png

4. Open index.html from the PARENT folder (empty-window), not from inside assets.

5. Press Ctrl + F5 to refresh the browser.

TIP: In File Explorer, turn ON "File name extensions" (View menu)
     so you don't accidentally save as "photo.png.png"

Your real photos will replace the colored placeholders automatically.
